# Installing Cursor

## macOS Installation

1. Visit [cursor.sh](https://cursor.sh) and click the "Download" button
2. Open the downloaded `.dmg` file
3. Drag the Cursor app to your Applications folder
4. Launch Cursor from your Applications folder
5. If you see a security warning, go to System Settings â†’ Privacy & Security and click "Open Anyway"

On first launch, Cursor will prompt you to:
- Sign in or create an account
- Migrate settings from VS Code (optional)
- Configure your AI model preferences
